# **Small airplane form**
This project represents, on a smaller scale, along with other projects, what will be one of the next ones, with management and organization. From this, you can see what is needed for an airplane ticket, where you have to fill in the following fields like first name, last name, the destination which can be changed, the number of tickets with the price written next to the destination, the departure and return dates, and optional extra services, and by clicking send, the total is calculated.
